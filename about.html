About Me

I am a software engineer with years of experience working in fast-paced, Agile (Scrum and Kanban) environments. I am very experienced with developing and designing on web platforms.

I'm proficient with C#, PHP, Python, JavaScript, MVC 3 with Razor, jQuery, and ASP.NET. I have experience with a number of JavaScript libraries. I have worked with a many third party controls including Dundas, Telerik, and DevExpress. I have recently been working with several front end tools including Twitter Bootstrap, datatables.net, and Handlebars to create modern web apps. This site was created using these very technologies.

After using the revealing module pattern for JavaScript for a while, I found myself drawn to TypeScript. I have become something of a TypeScript evangelist now that I have worked with it for a while.

I have worked with several relational database systems, including MS SQL Server 2005 and 2008, Postgres, and MySQL. Have additionally worked with configuring and integrating AppFabric caching (aka Velocity).

Additionally, I am familiar with graphic design tools including Adobe Photoshop, Flash, Adobe Premiere, Final Cut Pro, Illustrator, and Motion. Design is in my heart.

In my free time I often write programs in Python and would like to create iOS and Android mobile apps soon.

I currently reside in Atlanta with my dog and use the Dvorak layout to type.
